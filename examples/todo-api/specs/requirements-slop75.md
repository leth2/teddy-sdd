# 요구사항: todo-api (Slop≈75)

## 개요
TODO를 관리하는 API 서버를 만든다.

## 요구사항

### 1. 인증

**1.1** 사용자 로그인 기능을 구현한다.
- JWT 토큰을 사용한다.
- 잘못된 자격증명이면 401을 반환한다.

**1.2** 인증된 사용자만 TODO API를 사용할 수 있어야 한다.

### 2. TODO 관리

**2.1** TODO를 생성할 수 있어야 한다.

**2.2** TODO 목록을 조회할 수 있어야 한다.
- 필요한 경우 필터링을 지원한다.

**2.3** TODO를 수정할 수 있어야 한다.
- 수정 권한은 적절히 제한한다.

**2.4** TODO를 삭제할 수 있어야 한다.

### 3. 태그

**3.1** TODO에 태그를 붙일 수 있어야 한다.

**3.2** 태그로 TODO를 분류할 수 있어야 한다.
