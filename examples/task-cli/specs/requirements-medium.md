# 요구사항: task-cli (Slop≈51)

## 개요
로컬 CLI TODO 관리 도구.

## 요구사항

**REQ-001** `task add "<title>"` 명령으로 태스크를 추가한다.
- 제목 필수, 최대 200자.
- 성공 시 exit 0.

**REQ-002** `task list` 명령으로 태스크 목록을 조회한다.
- 태그, 상태 필터링 지원.

**REQ-003** `task done <id>` 명령으로 태스크를 완료 처리한다.
- 존재하지 않는 ID 처리 필요.

**REQ-004** `task delete <id>` 명령으로 태스크를 삭제한다.

**REQ-005** `task show <id>` 명령으로 태스크 상세 정보를 표시한다.

**REQ-006** 알 수 없는 서브커맨드는 오류를 반환한다.
